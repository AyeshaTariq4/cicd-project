
# CI/CD Pipeline Deployment on AWS using Docker & Terraform 🚀

## Overview

This project demonstrates a complete CI/CD pipeline for deploying a Flask application on AWS EC2 using Docker, DockerHub, GitHub Actions, and Terraform.

The workflow automates the process from code commit to application deployment, reducing manual effort and ensuring consistent deployments.

---

## Architecture

Developer → GitHub Repository → GitHub Actions → Docker Build → DockerHub → AWS EC2 → Live Application
